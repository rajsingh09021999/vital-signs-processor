About This Tool

What This Tool Does

This web application processes medical simulation data from ZOLL devices, automatically organizing vital signs data into structured Excel files based on simulation schedules and mannequin assignments.

Data Processing Pipeline

File Extraction
Recursively finds all JSON files in uploaded folders and subfolders
Data Parsing
Extracts vital signs from ZOLL JSON format including HR, BP, SpO2, CO2, etc.
Classification
Determines course, simulation, and mannequin based on timestamps and device serials
Organization
Creates structured folder hierarchy and Excel files for each simulation session

Supported Vital Signs

  • Heart Rate (HR)
  • Blood Pressure (NIBP/IBP)
  • Oxygen Saturation (SpO2)
  • Temperature
  • Respiratory Rate
  • CO2 Levels (FiCO2/EtCO2)
  • Hemoglobin (SpHb)
  • Perfusion Index (PI)

Privacy & Security

All files are processed locally on the server and are automatically deleted after processing. No data is permanently stored.
