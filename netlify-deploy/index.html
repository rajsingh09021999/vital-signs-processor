About This Application

What This Tool Does

This advanced web application processes medical simulation data from ZOLL devices, automatically organizing vital signs data into structured Excel files based on simulation schedules and mannequin assignments.

How Files Are Sorted & Organized

Automatic Classification: The system uses device serial numbers and timestamps to intelligently sort your data:

  • Course Assignment: 2025A through 2025N based on date ranges
  • Simulation Day: Sim1-Sim5 (Monday-Friday) calculated from course start dates
  • Mannequin Mapping: Dave, Chuck, Freddy, Matt, Oscar via device serial numbers
  • Output Structure: Course/Simulation/Mannequin/Date/ExcelFile.xlsx

Example Output: 2025A/Sim1/Dave/2024-10-14/2025A_Sim1_Dave_20241014.xlsx
